# TEX snippets CheatSheet

`<_>`: snippet argument, T: trigger key needed

## Environments

- `beg T-> begin end`
- `mc T-> \[ \]`
- `mi T-> \( \)`
- `M<p211n> -> \begin{pmatrix}1\\1\end{pmatrix}`
- `lr<)> -> \left( \right)`
- `sst -> \{ \}`
- `dsp -> \displaystyle`
- `ali T-> begin aligned`
- `gat T-> begin gathered`
- `cas T-> begin cases`
- `enm T-> begin enumerate`
- `itz T-> begin itemize`
- `adm T-> begin addmargin[1em][0em]`
- `mitb T-> begin minted{}`
- `miti T-> \mintinline{}{}`

## Auto

- `<x1> -> x_1`
- `<x11> -> x_{11}`
- `<1>/<2> -> \frac{1}{2}`
- `<3zpsq>/ -> \frac{3\pi^2}{}`
- `(<xsq - 7za>)/ -> \frac{x^2 - 7\alpha}{}`
- `(<3x - y>)rt -> \sqrt{3x -y}`
- `(<3x - y>)rt3 -> \sqrt[3]{3x -y}`
